# cinder-atlas

Adopted for personal usage by thyruh and Sartre. To quote the latter: "this theme is orgasmical".

# INSTALLATION


## Manual old fashioned way (I cannot be bothered to submit it to MELPA) ##
Download the theme to your local directory. You can do it through `git
clone` command:

```
git clone https://github.com/thyruh/cinder-atlas-theme.git
```

Then add path to cinder-atlas-theme to custom-theme-load-path list —
add the following to your emacs config file somewhere (.emacs,
init.el, whatever):

```
(add-to-list 'custom-theme-load-path
             "/path/to/cinder-atlas-theme/")
```

Use `M-x customize-themes` or `M-x load-theme` to change your current theme.
